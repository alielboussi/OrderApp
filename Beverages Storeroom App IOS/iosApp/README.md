# iOS App (SwiftUI)

## Setup on macOS
1. In Xcode, create a new iOS App project named **BeveragesStoreroomAppIOS** inside this `iosApp` folder.
2. Replace the generated Swift files with the contents of `Source/`.
3. Set deployment target to iOS 15+.
4. Run `pod install` and open the `.xcworkspace`.

## Supabase + Telegram config
Set credentials in `Source/Services/AppSecrets.swift` before running.
